无限的二维平面被划分成了若干个三角形区域。该平面的密铺方式定义如下（结合附图可以更好地理解）：

  • 根据欧拉公式（对于实数 $x$，有 $e^{ix}=\cos(x)+i\sin(x)$），首先，对于所有整数 $x, y$，在复平面上的 $x+y\exp(\pi i/3)$ 处绘制一个顶点。
  • 接着，对于上述步骤中构成边长为 $1$ 的等边三角形的任意三个顶点，连边形成三角形的边界。此外，在每个等边三角形的中心绘制一个顶点，并将该中心顶点与三角形外围的三个顶点分别连边。

给定 $N$（$2\le N\le 2\cdot 10^5$）个输入点，每个点都严格位于某个三角形区域内（即不在任何顶点或边上）。对于任意一对输入点，它们之间的**距离**定义为：连接这两点且不经过任何顶点的路径，所需要穿过的最少边数。

请输出所有 $N(N-1)/2$ 对输入点之间的距离之和。

输入格式（终端 / stdin）：

第一行包含一个整数 $T$（$T\ge 1$），表示独立测试用例的数量。每个测试用例的格式如下：

第一行包含一个整数 $N$。

接下来的 $N$ 行，每行包含三个整数 $x, y, z$（$0\le x,y<10^6, 0\le z<12$），表示复平面上的一个点 $x+y\exp(\pi i/3)+ \epsilon\cdot \exp((1+2z)\pi i/12)$（其中 $\epsilon$ 是一个极小的正数）。

保证所有测试用例中 $N$ 的总和不超过 $2\cdot 10^5$。

输出格式（终端 / stdout）：

对于每个测试用例，在新的一行中输出所有 $N(N-1)/2$ 对点之间的距离之和。

输入样例：

6
2
0 0 0
0 0 0
2
0 0 0
1 1 7
2
0 0 0
0 0 6
3
0 0 1
0 0 5
0 0 9
2
0 2 11
1 1 1
2
2 0 11
1 1 1

输出样例：

3
6
12
2
6
